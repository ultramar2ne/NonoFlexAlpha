#NonoFlexAlpha

## 노노유통
- NonoFlex 프로젝트는 '화성시니어클럽'의 노노유통 사업의 유통 시스템을 제작하는 프로젝트입니다.
- 어르신들도 쉽게 사용할 수 있는 유통 플랫폼을 목표로 개발을 진행했습니다.
- NonoFlexAlpha 프로젝트는 android로 구현된 0.9버전 앱의 확장판입니다.
- 여러 환경에 대응하기 위해 Flutter로 제작되었고, 배포는 배포사이트를 통해 진행됩니다.
